Develop an Oracle Cloud Fixed Scope Offering Category 1 to publish on the cloud market place : APPLICATIONS for Oracle Cloud SaaS, PaaS or IaaS services. APPLICATIONS must be a generally available product, offering or cloud service that runs on/or integrates with Oracle Cloud. Listings for Applications will complete a technical review and live demonstartion running on the Oracle Cloud. Plus you will need a fully executed Cloud Interoperability Addendum in addition to the OPN Agreement. Category 2 to publish on the cloud market place: SERVICES for Oracle Cloud SaaS, PaaS or IaaS services. To add details of any "Oracle Cloud " solutions to the SERVICES listing section of the Oracle Cloud Marketplace, a successful customer deployment/reference is required.
