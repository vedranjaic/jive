Overview

A Fixed Scope Offering (FSO) is a professional services package for the implementation of an Oracle Cloud solution.

FSOs are created by OPN members, reviewed and accepted by Oracle, and showcased to customers on the Oracle Cloud Marketplace and Oracle Solutions Catalog.
