Implementation Exams

Implementation Exams

Implementation certifications (exams) are one of the key competency requirements for your Company's OPN specialization. Implementation Specialist exams are only delivered with a proctor through Pearson VUE testing centers or special testing events organized by Oracle and called Test Fests. To attend the exam you need to have a special voucher and active Pearson VUE Web Account.
