Cloud Partner Program


Oracle PartnerNetwork (OPN) Cloud designations offer partners the opportunity to showcase expertise, skills and investment in the Oracle Cloud. Oracle's Cloud Programs Recognize and Reward Investments and Expertise in Oracle Cloud. 

The new Cloud program, with tiered designations, recognizes those partners that engage with Oracle and invest in Oracle Cloud, by offering them incremental and progressive benefits to complement and build on the existing OPN program levels. Partners are now able to differentiate their Oracle Cloud expertise and success with customers

Qualification into these tiers is based on elements that exemplify excellence in Oracle Cloud solutions. For more information and to review the selection criteria for each Tier,  please visit the Cloud  Designations Brochure

Fixed Scope Offerings 


What is a Fixed Scope Offering?


A Fixed Scope Offering (FSO) is a professional services package for the implementation of an Oracle Cloud solution. FSOs are created by OPN members, reviewed and accepted by Oracle, and showcased to customers on the Oracle Cloud Marketplace and Oracle Solutions Catalog.
